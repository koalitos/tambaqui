🐟 Tambaqui

IA brasileira de código. Roda local, API OpenAI-compatível, baixa modelos HuggingFace, admin web.

Open Source API OpenAI HuggingFace Docker Linux/macOS
Instalar Agora GitHub

O que é

API OpenAI

Compatível com qualquer client que use a API da OpenAI. Continue.dev, Open WebUI, LibreChat, etc.

Modelos HuggingFace

Baixe modelos de código (Qwen, DeepSeek, CodeLlama) direto pelo admin web.

Pre-loaded

Modelo fica quente na RAM como shaders de jogo. Warmup automático no boot.

Busca Web

Pesquisa em Wikipedia, StackOverflow, documentações oficiais e DuckDuckGo.

Admin Web

Gerencie modelos, users, sessões e monitore CPU/RAM pelo browser.

Streaming

Respostas aparecem token a token em tempo real. Chat web e API.

Instalar

Linux/macOS
Docker
Unraid
CasaOS

1. Clonar e instalar

git clone https://github.com/koalitos/tambaqui.git
cd tambaqui
bash install.sh

O instalador vai:

  • Instalar Python se necessário
  • Criar venv e instalar dependências
  • Baixar modelo leve (Qwen 0.5B - ~1GB)
  • Pedir pra criar o admin

2. Rodar

./tambaqui

Acesse http://localhost:8000

3. Chat no terminal

./tambaqui chat

Acesse /admin e clique "Baixar" no modelo desejado.

Opção 1: Template XML (recomendado)

No Unraid, vá em Docker → Add Container → Template URL e cole:

https://raw.githubusercontent.com/koalitos/tambaqui/main/unraid-template.xml

Clique Apply. O Unraid configura tudo automaticamente.

Opção 2: Docker Compose

mkdir -p /mnt/user/appdata/tambaqui
cd /mnt/user/appdata/tambaqui
wget https://raw.githubusercontent.com/koalitos/tambaqui/main/docker-compose.yml
docker compose up -d

Opção 3: Manual

docker run -d \
  --name tambaqui \
  -p 8000:8000 \
  -v /mnt/user/appdata/tambaqui/modelos:/app/modelos \
  -v /mnt/user/appdata/tambaqui/dados:/app/dados \
  -e TAMBAQUI_ADMIN_USER=admin \
  -e TAMBAQUI_ADMIN_PASS=tambaqui \
  --restart unless-stopped \
  ghcr.io/koalitos/tambaqui:latest

Acessar

Abra http://ip-do-unraid:8000

API OpenAI

Funciona com qualquer tool que suporte a API da OpenAI. Só trocar a base URL:

export OPENAI_API_BASE=http://localhost:8000/v1
export OPENAI_API_KEY=seu-token-do-login

Exemplo curl

curl http://localhost:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer seu-token" \
  -d '{
    "messages": [{"role": "user", "content": "crie um hello world em python"}],
    "stream": true
  }'

Endpoints

MétodoEndpointDescrição
GET/v1/modelsListar modelos
POST/v1/chat/completionsChat (com streaming)
POST/api/chatChat Tambaqui (com busca web)
POST/api/auth/loginLogin (retorna token)
GET/api/statusStatus do sistema
